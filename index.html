ListiFy

Turn a group of lines into an HTML list

  How it works

Just use some of this patterns to customize your list

  • li(id='item_#', class='list') > span(class='inner')  Apply
  • li(id='item#')  Apply
  • li() > h1(class='title') > span(class='inner')  Apply
  • li() > a(class='link', href='http://www.google.com')  Apply

1  If no Format is specified, you get a simple list.
2  Use Spaces to specify the indentation (4 default).
3  Ignore digits will ignore the first digits.
4  Pretty enable the pretty output.
5  Use Counter start and # to generate a numeric sequence.
